# Eliyah Webdesign Portfolio

**Moderne websites die voelen als een merk, niet als een template.**

Dit is mijn professionele portfolio website, gebouwd met moderne webtechnologieën en geoptimaliseerd voor snelheid, toegankelijkheid en gebruikerservaring.

## 🚀 Live Demo

Bekijk de website: [eliyah-webdesign.github.io](https://eliyah-webdesign.github.io) (wanneer gehost)

## ✨ Kenmerken

- **Dark Theme Design** - Moderne, professionele uitstraling
- **Volledig Responsive** - Perfect op desktop, tablet en mobiel
- **Smooth Animations** - Subtiele reveal-animaties voor betere UX
- **SEO Geoptimaliseerd** - Meta tags, semantische HTML, snelle laadtijden
- **Toegankelijk** - WCAG compliant met keyboard navigation en screen reader support
- **Performance First** - Lazy loading images, geoptimaliseerde fonts

## 🛠️ Technologieën

- **HTML5** - Semantische structuur
- **CSS3** - Custom properties, Flexbox, Grid, Animations
- **Vanilla JavaScript** - ES6+ features, Intersection Observer API
- **Google Fonts** - Inter font family voor professionele typografie
